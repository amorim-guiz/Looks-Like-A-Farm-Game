# Looks-Like-A-Farm-Game
"LOOKS LIKE A FARM GAME I'VE ALREADY PLAYED" é (provavelmente) um jogo de fazenda simples feito no Pico-8 utilizando uma série de videos-tutorial do canal SpaceCat

Esse projeto nada mais é que eu me forçando a tentar evoluir no desenvolvimento de jogos de uma vez por todas, aprendendo com vídeos de "code with me" do youtube e tentando adicionar meus prórprios traços no projeto.



Sobre o formato:

Os arquivos são carts.p8 que podem ser baixados e utilizados na versão de browser do pico-8, basta pesquisar a versão universitária do pico-8 no google, ou entrar nesse link:

https://www.pico-8-edu.com/

Baixe o arquivo, de play no site, arraste o arquivo para o console e pronto! Já pode testar, observar, acompanhar e editar, faça como preferir...

Existem diversos tutoriais sobre como utilizar o pico-8 no youtube, mas aqui estão algumas dicas:

Ao arrastar o arquivo, não se intimide pelo console, a tecla ESC do teclado te deixa explorar os editores (código, sprites, tileset, map, audio, sfx e etc...).

Se você não quiser se aprofundar nessas coisas, apenas digite RUN e de ENTER no console que ele irá tentar rodar o programa (existem versões dos arquivos 'llafg.p8' que não possuem código, escolha uma versão correspondente para testes adequados).



Sobre o esquema de versionamento:

Eu vou usar um esquema de versionamento simples no nome de cada arquivo, onde os números no nome demonstram as principais mudanças entre os arquivos anteriores, exemplo:

O arquivo 'llafg.p8' não tem nenhum número por ser só um arquivo raiz do projeto, já o arquivo 'llafg01.p8' apresenta mudanças na área visual de pixelart, tileset e mapa.

Para critério de entendimento, o esquema que eu seguirei será majoritáriamente de tal forma,

O primeiro número será relacionado ao desenvolvimento visual, o segundo número será relacionado ao desenvolvimento do código em geral e o terceiro número relacionado ao áudio em geral.

O arquivo milkshake01.02.03 por exemplo, estaria em sua primeira versão visual, mas ja teria duas versões de códigos e três versões com modificações em áudio.
