# Looks-Like-A-Farm-Game
"LOOKS LIKE A FARM GAME I'VE ALREADY PLAYED" é (provavelmente) um jogo de fazenda simples feito no Pico-8 utilizando uma série de videos-tutorial do canal SpaceCat

Esse projeto nada mais é que eu me forçando a tentar evoluir no desenvolvimento de jogos de uma vez por todas, aprendendo com vídeos de "code with me" do youtube e tentando adicionar meus prórprios traços no projeto.
